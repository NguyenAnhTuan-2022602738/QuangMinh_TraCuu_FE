LOGO SETUP INSTRUCTIONS
========================

Để hiển thị logo công ty, vui lòng:

1. Copy file logo (PNG format, recommended size: 150x150px or higher)
2. Đặt tên file là: logo.png
3. Save vào thư mục này (client/public/)

Hiện tại Header đang sử dụng fallback text "QM" cho đến khi logo.png được thêm vào.

File path expected: /logo.png (được serve từ public folder)
